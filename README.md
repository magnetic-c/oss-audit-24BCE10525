## 📌 Project Overview

This project is based on the study of open-source software using Git as an example. It covers the origin, license, ethical aspects, Linux usage, and ecosystem of Git.  

Along with theory, five shell scripts are created to demonstrate basic Linux commands and automation.

---

## 🧾 Scripts Included

### Script 1 — System Identity Report
Displays system details like kernel, user, uptime, and date.

### Script 2 — FOSS Package Inspector
Checks whether Git is installed and shows version details.

### Script 3 — Disk and Permission Auditor
Shows permissions and sizes of important system directories.

### Script 4 — Log File Analyzer
Reads a log file and counts occurrences of a keyword.

### Script 5 — Manifesto Generator
Takes user input and generates a simple open-source manifesto.
